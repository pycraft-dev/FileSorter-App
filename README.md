# 📂 FileSorter App (v2.0)
**Автоматический сортировщик файлов на Python с современным GUI (CustomTkinter)**  
Быстро наводит порядок в папке: распределяет файлы по категориям, поддерживает рекурсивный обход, режимы «переместить/копировать», защиту от перезаписи и логирование.

---

## 🚀 Установка

### Требования
- Python **3.11+**
- Windows 10/11 (на других ОС запуск возможен, но сборка в EXE — только Windows)

### Установка из исходников

```bash
git clone https://github.com/pycraft-dev/FileSorter-App.git
cd FileSorter-App

python -m venv .venv
.venv\Scripts\activate

pip install -r requirements.txt
python main.py
```

## 📘 Документация

### Как пользоваться
1. Запустите приложение: `python main.py`
2. Выберите папку для сортировки (входную директорию).
3. Выберите режим:
   - **Move** — переместить файлы в папки категорий
   - **Copy** — скопировать файлы в папки категорий (оригиналы останутся на месте)
4. Нажмите старт и дождитесь завершения. Результат — структура папок по категориям внутри выбранной директории (или в целевой папке, если она предусмотрена интерфейсом).

### Категории и расширения
Ниже — базовая таблица расширений. При необходимости список можно расширять в коде (см. раздел «Архитектура»).

| Категория | Расширения (пример) |
|---|---|
| **Photos** | `.jpg`, `.jpeg`, `.png`, `.gif`, `.bmp`, `.webp`, `.raw` |
| **Documents** | `.pdf`, `.doc`, `.docx`, `.txt`, `.rtf`, `.odt`, `.xls`, `.xlsx`, `.ppt`, `.pptx` |
| **Videos** | `.mp4`, `.avi`, `.mkv`, `.mov`, `.wmv`, `.flv`, `.webm` |
| **Audio** | `.mp3`, `.wav`, `.flac`, `.aac`, `.ogg`, `.wma` |
| **Archives** | `.zip`, `.rar`, `.7z`, `.tar`, `.gz` |
| **Other** | всё остальное |

### Безопасность
Приложение старается быть «безопасным по умолчанию»:
- **Защита от перезаписи**: если имя файла уже существует в папке назначения, применяется авто‑переименование.
- **Предупреждение для системных путей**: при попытке сортировки системных директорий показывается предупреждение (во избежание повреждения ОС).
- **Игнорирование технических файлов**: системные/служебные элементы могут пропускаться (зависит от реализации в сортировщике).

## 🏗 Архитектура

Проект построен модульно: UI отделён от логики сортировки и утилит.

```
FileSorter-App/
├── main.py                  # Точка входа
├── src/
│   ├── core/                # Бизнес-логика сортировки
│   ├── ui/                  # Окна и компоненты CustomTkinter
│   └── utils/               # Вспомогательные функции
├── requirements.txt
├── LICENSE
└── README.md
```

Где обычно лежит главное:
- **Категории/расширения**: `src/core/file_types.py` (или аналогичный модуль)
- **Сортировщик**: `src/core/file_sorter.py`
- **Главное окно**: `src/ui/main_window.py`

## 🧰 Сборка в EXE (Windows)

### Вариант 1 — PyInstaller (рекомендуется)

```bash
pip install pyinstaller
pyinstaller --onefile --windowed main.py
```

Результат:
- EXE появится в папке `dist/`

Полезные флаги (по желанию):
- `--name FileSorterApp` — имя файла
- `--icon images/app.ico` — иконка (нужен `.ico`)

### Вариант 2 — запуск без сборки
Если EXE не нужен, достаточно `python main.py` из виртуального окружения.
